# Java Calculator

## 프로젝트 소개
STEP2 까지 진행한 계산기입니다.
양의 정수 2개와 사칙연산 기호를 입력하여 계산 결과를 출력해주는 계산기 입니다.

## 주요 기능
- 양의 정수 입력 받기 (첫번째, 두번째)
- 사칙연산(+,-,*,/) 수행
- 나눗셈 연산일 때 두번째 정수에 0이 올 경우 경고 출력 후 처음부터 재입력할 수 있게 하는 부분
- 계산 결과를 저장할 수 있다.
- 저장된 결과를 조회할 수 있다.
- 가장 오래된 저장 결과를 삭제할 수 있다.
- exit 입력 시 종료

## 실행 방법
1. App Class 실행
2. 첫 번째 숫자를 입력한다.
3. 사칙연산 기호를 입력한다.
4. 두 번째 숫자를 입력한다.
5. 결과 확인
6. 결과 확인 후 **LIST 조회 가능** OR **Exit** 입력 후 종료

## 파일 구조
- 'App.java' : 입력, 출력, 반복문 처리
- 'Calculator.java' : 계산 기능, 결과 저장 및 삭제 기능

## 예외 처리
- 음수는 입력이 불가하다.
- 사칙연산 기호 외 입력이 불가하다.
- 나눗셈에서 두 번째 숫자에 0 입력이 불가하다.
